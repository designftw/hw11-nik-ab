Animations for chat app

Here are the animations that I added to my chat app. They are not very disruptive and they improve feedback of the website.

  1. Like button

    When user likes a message, the like button does a bounce animation

  2. Message appear/disappear

    Whenever a new message appears or disappears (when someone sends or deletes a message), The missage will have scaling animation.

  3. Delete confirmation

    When a user clicks delete button confirmation appears (and disappears) with animation.

  4. Read receipts

    When a user hovers over the number of people that saw the message, a popup appears with animation, that displays the users that saw the message.

  5. Reply list

    When a user clicks view/hide replies, reply list appears with animation.

  6. Profile pic loading

    While a profile picture is loading, it is replaced by a blinking animation to show that the picture is loading.
